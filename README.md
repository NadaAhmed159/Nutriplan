# 🥗 NutriPlan — Smart Nutrition & Meal Planning Platform

NutriPlan is a modern nutrition and meal planning web application that helps users discover healthy meals, analyze nutritional information, and make informed dietary decisions through an intuitive and responsive interface.

The application integrates multiple REST APIs to provide real-time meal data, nutrition analysis, recipe search, barcode-based food lookup, and personalized meal planning features.
